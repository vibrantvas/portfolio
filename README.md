## 📝 About This Project

This is my personal portfolio website built from scratch to showcase my skills, projects, and professional experience as a Systems Integration Specialist and AI Management Expert. It features a clean, modern design with smooth animations, full responsiveness, and professional presentation.

**Purpose:** Demonstrate frontend development skills while providing a professional online presence for job opportunities and client acquisition.

## ✨ Features

- ✅ **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- ✨ **Smooth Animations** - Scroll-triggered animations for engaging user experience
- 🎯 **Smooth Scrolling** - Navigation that smoothly scrolls to page sections
- 📧 **Contact Form** - Working contact form with validation
- 🎨 **Modern Design** - Clean, professional aesthetic with attention to detail
- ⚡ **Fast Loading** - Optimized for performance with minimal load time
- ♿ **Accessible** - Built with accessibility best practices (ARIA labels, semantic HTML)
- 📱 **Mobile-First** - Designed for mobile and scaled up to desktop

## 🛠️ Built With

- **HTML5** - Semantic markup for better SEO and accessibility
- **CSS3** - Modern styling with Flexbox and Grid layouts
- **JavaScript** - Vanilla JS for interactions (no frameworks needed)
- **GitHub Pages** - Free hosting for static sites
- **Google Fonts** - Professional typography (Inter, Poppins)
- **Font Awesome** - Icons throughout the site

## 🚀 Quick Start

### View Locally

```bash
# Clone the repository
git clone https://github.com/vibrantvas/portfolio.git

# Navigate to directory
cd portfolio

# Open in browser
open index.html

# OR use a local server
python -m http.server 8000
# Visit: http://localhost:8000
```

### Deploy to GitHub Pages

1. Push your code to the repository
2. Go to Settings → Pages
3. Select `main` branch and `/ (root)` folder
4. Click Save
5. Your site will be live in a few minutes
